# **🌷 이미지 캡셔닝 게임**

------

## **💜 프로젝트 소개** 

---

점차 발전하고 있는 인공지능(AI)을 이용한 이미지 캡셔닝 기능을 게임에 접목하여 유저들에게 좀 더 혁신적이고 새로운 패러다임을 제공하고자 함



## **✔ 사용스택**

------

**Backend**

- Spring Framework 4.0.5

**Frontend**

- React Native
- MUI

AI

- PyTorch
- GPT-2 ko
